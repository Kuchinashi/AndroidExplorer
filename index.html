Quick Documents

Key Description
F2 Rename the selected item.
F5 Refresh file list.
Del Delete the selected item(s).
Context Menu
Refresh file list.
Download the selected item(s).
Delete the selected item(s).
Rename file
Drag and Drop
Drag and Drop the file(s) or folder(s) of Windows, to push the current location of the Android Explorer.
Double click list item
Open in the application associated with the file in Windows. After you close the application by overwriting save, you can push to the android device.
Text box in the lower right
Move directly to the directory.

Quick Documents - ja

キー 概要
F2 選択されたファイル/フォルダの名前を変更します．
F5 ファイル一覧を更新します．
Del 選択されたファイル/フォルダを削除します．（複数可）
右クリックメニュー
ファイル一覧を更新します．
選択されたファイル/フォルダを指定のフォルダへダウンロードします．（複数可）
選択されたファイル/フォルダを削除します．（複数可）
選択されたファイル/フォルダの名前を変更します．
ドラッグアンドドロップ
WindowsのエクスプローラからAndroid Explorerへファイルやフォルダをドラッグ＆ドロップすると，端末へ転送します．
リストのファイルをダブルクリック
Windowsでそのファイルに関連づけられたアプリケーションで開きます．編集後，上書き保存してアプリケーションを閉じると，そのまま端末にpushすることもできます．
右下のテキストボックス
入力されたAndroid側のパスへダイレクトに移動します．
